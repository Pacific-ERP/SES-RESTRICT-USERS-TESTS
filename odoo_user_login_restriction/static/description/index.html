Odoo User Login Restriction


Restrict Users to Access their Accounts in Odoo!

Odoo User Login Restriction allows the users to login to your Odoo from certain IP Addresses only. Select the allowed access right groups to access your Odoo. Easily, set the user login restriction for the security of the account.

Restrict User Access Via IP Address

Restrict user access except the IP Addresses that are mentioned /allowed in Odoo Backend.

Add the permitted IP addresses in the settings and the user can login with those IP Addresses only.

It restricts the access of user account by IP.

Hence the user can login to their account by some specified IP addresses only.

Allow Users With Mentioned Access Rights Only

Allow users with certain allowed rights to login to your Odoo.

Add the access rights in Odoo Backend so that only users with mentioned access rights can login to the database.

If the user is following any of the condition then he/she can login.

Set Up Maximum Login Attempts to Odoo

Add the maximum number of attempts allowed to login. It is a great step for the safety of your login credentials.

Freeze the login page once the user has reached the maximum attempts for login.

Set the cooldown time in seconds once the login page freezes due to wrong login attempts.
